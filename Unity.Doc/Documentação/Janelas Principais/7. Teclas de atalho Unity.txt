Teclas de atalho Unity
Esta página fornece uma visão geral dos atalhos de teclado padrão do Unity. 
Você também pode baixar um PDF da tabela para Windows e MacOSX. 
Quando um comando tem Ctrl / Cmd como parte do pressionamento de tecla, isso indica que a Control chave deve ser usado no Windows e na tecla Command no MacOSX.
O trackpad do Mac também possui vários atalhos para navegar na visualização de cena.
Consulte Navegação na vista da cena para saber mais sobre isso.

Ferramentas

W								Mover

GameObject

Ctrl / Cmd + Shift + F			Alinhar com vista
